## ğŸ“Š Workflow Overview

### 1. Data Enrichment (Task 1)

* Load raw dataset and reference codes
* Validate dataset schema
* Add new **observations** (smartphone penetration, internet usage, etc.)
* Add new **events** (regulations, policy changes)
* Add **impact links** between events and indicators
* Save enriched dataset and enrichment log

### 2. Exploratory Data Analysis (Task 2)

* Dataset overview and quality assessment
* Temporal coverage & gaps
* Pillar-based distribution and access analysis
* Usage, infrastructure, and event trend visualization
* Correlation analysis & key insights
* Identify sparse or new indicators
